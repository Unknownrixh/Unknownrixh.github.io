# 🌐 [Day-3] 네트워크 기초: IP와 TCP 분석

클라우드 보안 엔지니어가 되기 위한 필수 기초, 컴퓨터 간의 통신 규칙인 TCP/IP에 대해 정리해 보았습니다.

## 1. IP (Internet Protocol)
- **정의:** 컴퓨터의 고유한 **주소**입니다.
- **역할:** 인터넷상에서 택배를 보낼 때 '도착지 주소'를 적는 것과 같은 역할을 합니다.

## 2. TCP (Transmission Control Protocol)
- **정의:** 데이터를 목적지까지 안전하고 정확하게 보내기 위한 **배송 시스템**입니다.
- **역할:** 데이터가 들어오면 그 데이터를 작은 단위로 **쪼개서** 보냅니다.
- **핵심:** 데이터의 **손실을 최소한으로 줄이고**, 목적지에 도착한 데이터를 원래 순서대로 꼼꼼하게 조립합니다.

> 💡 **보안 엔지니어의 관점 한 스푼!**
> TCP는 데이터가 '안전하게 도착'하도록 보장해 주지만, 누군가 중간에 훔쳐보는 '데이터 유출'을 막아주지는 않습니다. 따라서 실제 클라우드 환경에서는 데이터 유출을 막기 위해 **암호화 기술(HTTPS, TLS 등)**을 추가로 적용해야 합니다.
